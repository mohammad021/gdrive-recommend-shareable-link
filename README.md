# GDrive: Recommend Shareable Link

**اسکریپت Tampermonkey برای ساخت لینک‌های اشتراک گوگل‌درایو و اعمال فونت وزیرمتن**

![GitHub](https://img.shields.io/badge/Tampermonkey-Required-brightgreen)
![Version](https://img.shields.io/badge/Version-1.2-blue)

## امکانات

- **تشخیص هوشمند ID** فایل از URL و محتوای صفحه
- نمایش جعبه **لینک‌های پیشنهادی** (به خصوص لینک مناسب برای RapidLeech)
- دکمه **کپی سریع** برای هر لینک
- اعمال خودکار **فونت وزیرمتن** روی صفحات هشدار و دانلود گوگل‌درایو
- منوی Tampermonkey برای ساخت لینک از ID یا URL دلخواه
- پشتیبانی کامل از انواع لینک‌های گوگل‌درایو
- **بروزرسانی خودکار**

## نحوه نصب

1. افزونه **Tampermonkey** را در مرورگر خود نصب کنید.
2. روی لینک زیر کلیک کنید:

**[📥 نصب مستقیم از GitHub](https://raw.githubusercontent.com/mohammad021/gdrive-recommend-shareable-link/main/GDrive-Recommend-Shareable-Link.user.js)**

3. در پنجره باز شده، **Install** را بزنید.

## نحوه استفاده

- به طور خودکار روی صفحات دانلود و هشدار گوگل‌درایو فعال می‌شود.
- جعبه لینک‌ها در پایین صفحه نمایش داده می‌شود.
- از منوی Tampermonkey (راست کلیک روی آیکون) می‌توانید لینک بسازید.
